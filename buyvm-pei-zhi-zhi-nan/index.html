Buyvm增加存储块及配置指南

01-17-2023 2 min read

准备：

  1. 服务器盒子 1G内存
  2. 存储块 512G

登陆buyvm管理台
登陆地址：https://manage.buyvm.net/（这里建议google浏览器，别问我为什么，到时候你就知道了）

这里存储块和服务器就绑定好了，如果需要重装系统，

装完系统就可以直接使用ssh客户端连接安装qb了

打开cmd，输入ssh连接命令ssh [email protected]你的服务器IP

登陆后，直接输入下面命令安装杰佬一键脚本

bash <(wget -qO- https://raw.githubusercontent.com/jerry048/Dedicated-Seedbox/main/Install.sh) user password 1

# user 用户名自己取，password 密码自己设，最后一位是QB缓存大小 (GB), 大佬推荐设为系统内存的1/4到1/8，我选的盒子是32G，设置了8GB

然后连续2个N

直到显示BBR这里一定要输入Y

默认回车

然后安装vnstat

apt update

apt install vnstat

reboot

等待重启完成后，直接浏览器输入IP地址：8080就可以进入qb了，用户名密码为最前面代码中的用户名密码。

由于我这里是1G内存，比较小，所以QB还需要做一些优化

挂载硬盘到系统，ssh输入

fdisk -l

说明，常用的就是那几个，d n p w q，有这5个足够了

对新硬盘进行分区，输入：

fdisk /dev/sda

保存退出 按 w

分区完毕后对分区进行格式化：

mkfs.ext4 /dev/sda

挂载分区：

mkdir /data1

mount /dev/sda /data1

输入：

df -h

验证分区是否挂载成功

利用dd进行读写速度测试，输入：

dd if=/dev/zero of=/data1/test bs=1M count=10240

或者

bash <(wget -qO- https://bench.sh)

配置开机自动挂载

编辑“/etc/fstab”文件：

vim /etc/fstab

在最后加入一行

/dev/sda /data1 ext4 defaults 0 0

配置qb保存目录
